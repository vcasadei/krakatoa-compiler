Dúvidas
================================
  
Dúvida 1:  
O código  
	if ( expr ) statement;  
em Krakatoa, onde o tipo de 'expr' é boolean, deve ser traduzido para   
	if ( (expr) != false ) statement;  
em C.   
E o código   
	if ( ! expr ) statement;  
deve ser traduzido para  
	if ( (expr) == false ) statement;
